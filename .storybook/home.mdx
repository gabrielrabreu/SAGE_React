# SAGE UI

Welcome to **SAGE UI**, a library of reusable React components designed to streamline UI development.

---

## Overview

SAGE UI is a collection of components built with React and TypeScript. It aims to provide a consistent and cohesive set of UI elements that can be easily integrated into your projects.

---

## Components

SAGE UI includes the following categories of components:

### 1. Atomic Components

These are the basic building blocks of the UI, such as buttons, inputs, and icons.

### 2. Form Components

Components specifically designed to facilitate form creation and input handling, including text fields, checkboxes, and select dropdowns.

### 3. Layout Components

Components for organizing and structuring your UI, including grids, containers, and navigation bars.

### 4. Data Display Components

Components for displaying data and information, such as tables, cards, and tooltips.

---

## Getting Started

To get started with SAGE UI, check out the documentation for each component category and explore the available options and variations.
